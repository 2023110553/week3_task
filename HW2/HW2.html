멋쟁이 사자처럼 프론트엔드 1주차 과제
어려웠던점
flex align items과 contents의 차이를 이해하는게 어려웠다.
grid 0부터시작하는 이차원배열이랑 그리드의 작동방식이 달라 헷갈렸다.
과제 추가로 과제를하며 padding과 margin의 차이를 직관적으로 이해하기 어려웠다.
flex
  • flex-direction
    -> 메인축 변경
  • justify-content
    -> 메인축 정렬
  • align-items
    -> 수직축 정렬
  • align-content
    -> 수직축 정렬한것끼리 정렬
  • flex-wrap
    ->여유 공간 없을때
grid
  • grid template
    -> 셀 크기 정하기
  • grid-auto
    -> 이후 생기는것들 기준
  • grid-auto-flow
    -> 이후 들어오는것들 정렬방식
